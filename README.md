# Empire Salon Website

This is the official website for **Empire Salon**, a hair salon in Palanpur, Gujarat.

## Live Website
👉 https://shlokbyte.github.io/trial-1/

## What's on the website
- Home page with the salon name and a booking button
- List of all services offered
- Gallery with real photos from the salon
- Contact section with phone number and address

## Files in this project
- `index.html` — the main website
- `haircut.jpeg` — haircut photo for the gallery
- `beard.jpeg` — beard styling photo for the gallery

## Tools used
- HTML, CSS, JavaScript
- GitHub Pages (for hosting)
